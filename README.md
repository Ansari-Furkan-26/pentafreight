🧠 Development Approach (README Content)
# Pentafreight Frontend Clone

## Approach

1. Initial Analysis
- Attempted to extract structured content using Firecrawl.
- Captured full-page visual references using browser extensions.


2. Layout Recreation
- Converted reference visuals into structured UI components.
- Maintained brand consistency (colors, typography, spacing).


3. Asset Handling
- Extracted and reused necessary assets from the reference website.
- Organized assets inside `public/media` for efficient access.


4. Animation
- Used Framer Motion for smooth, controlled animations.
- Focused on minimal and purposeful motion instead of heavy effects.


5. Iterative Development
- Followed an iterative approach:
  - Build → Review → Improve → Repeat
- Continuously refined UI for a more professional and polished look.


6. Responsiveness
- Designed with a mobile-first mindset.
- Ensured compatibility across all device sizes.


7. Deployment
- Deployed via Netlify connected to GitHub.
- Enabled automatic deployments on every push.


8. Routing
- Added redirect handling to ensure smooth navigation and page refresh behavior.
